| Design Token Overrides | <img src={JSON_Logo} width="42"/> | Converted to CSS | <img src={CSS_Logo} width="36"/> |
| Variable definitions | <img src={JSON_Logo} width="42"/> | Converted to Typescript | <img src={TS_Logo} width="36"/> |
